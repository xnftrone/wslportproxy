# AGENTS

[English Version](./AGENTS.en.md)

## 项目目标

本项目是一个 Windows 原生命令行工具，用于持续维护从宿主机到 WSL 发行版的 TCP 端口转发。

## 不可违反的安全规则

- 绝不能把清理范围扩大到当前命令未显式声明的端口。
- 绝不能批量删除无关的 `portproxy` 规则。
- 绝不能删除不属于本工具的防火墙规则。
- 如果宿主机上已有活动 TCP 监听器占用目标端口，必须拒绝修改该端口。
- 如果目标端口上已存在非本工具托管的 `portproxy` 规则，必须拒绝接管。
- 对不确定情况默认“拒绝修改”，不要强行覆盖。

## 实现要求

- 保持工具是明确面向 Windows 的实现。
- 保持前台日志输出和退出自动清理能力。
- 只在显式声明端口范围内工作。
- 命令行帮助内容变更时同步更新文档。
- 公开文档保持环境无关，不要写入个人机器路径或本地流程。
- 机器相关说明放在本地文件中，不放进公开 README。

## 测试要求

- 每个行为改动都应补充或更新单元测试。
- 必须验证参数解析、冲突检测和清理边界。
- 如果改动了端口安全逻辑，必须有对应通过的测试作为证据。
